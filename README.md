# 🛒 NovaShop - Mini E-commerce con Django

Proyecto de tienda online desarrollado con Django. Permite visualizar productos, agregarlos a un carrito y simular un proceso de compra.

## 🚀 Funcionalidades

- 🛍️ Listado de productos
- 🛒 Carrito de compras
- ➕➖ Control de cantidades
- ❌ Eliminar productos del carrito
- 💳 Checkout con formulario de pago simulado
- 📦 Flujo completo de compra

## 🧰 Tecnologías usadas

- Python
- Django
- HTML + CSS
- SQLite
